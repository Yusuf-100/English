# 作文百句

從國三到高三，老師提供了很多英文作文讓我們參考，並從中提取多個單字、片語、佳句，讓學生慢慢學習寫作文。雖然老師將作文分成多個段落，按段落編號，但我將作文重新整合起來，&#x4EE5;**`作文`**&#x4F86;編號，每50篇編成一頁以方便閱讀。
